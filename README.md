# Student Task Organiser

A clean and responsive web application designed to help students manage their academic tasks efficiently.
It allows users to organize assignments, deadlines, and daily study activities in one place.
The project features a modern and user-friendly interface for a smooth experience.
Students can keep track of pending and completed tasks with ease.
Built using HTML, CSS, and JavaScript with a focus on responsive design.
The website aims to improve productivity and simplify academic planning.
It demonstrates frontend development skills, including layout design and interactive features.
This project was created as part of my web development portfolio.
Future updates will include reminders, user authentication, and cloud data storage.
